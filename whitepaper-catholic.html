1. Mission & Vision

Vatican Bank (Catholic Branch) bridges ancient Catholic Christian traditions with modern blockchain technology. We provide a transparent, secure, and decentralized platform for spiritual acts: candles, prayers, donations, confession, and charity. Our mission is to make faith accessible to the digital generation while maintaining full respect for Catholic traditions.

2. Technology Stack

  • Blockchain: Solana (Mainnet)
  • Token Standard: Token-2022
  • Smart Contracts: Verified on Solscan
  • Frontend: HTML/CSS/JS, hosted on GitHub Pages
  • Transaction Verification: Solscan Public API

4. Core Mechanics

  • For God (За Бога): Send 1 DIABOLUS (VBDC) to DEUS (VBDT) address → light a candle (1 hour)
  • Against God (Против Бога): Send 1 DEUS (VBDT) to DIABOLUS (VBDC) address → appear on the "Sinners Wall"
  • Eternal Candle (Вечная свеча): Send 1 SANCTUS (VBST) + 1 DIABOLUS (VBDC) to DEUS (VBDT) address → eternal candle (never expires)
  • 40+ Spiritual Deeds: Various actions (prayers, memorials, donations, NFT icons, pilgrim certificates)
  • Confession: Anonymous confession with automatic answer
  • DAO: DEUS (VBDT) holders can vote on treasury allocation

5. Smart Contract Security

  • All token contracts are verified on Solscan
  • Mint and freeze authorities are revoked — no one can mint new tokens or freeze wallets
  • All transactions are recorded on Solana blockchain and publicly visible
  • No central control — the website only provides a convenient interface

6. Roadmap

  • Q1 2025: Token creation (DEUS, SANCTUS, DIABOLUS), website launch, basic mechanics
  • Q2 2025: 40 Spiritual Deeds, Confession page, DAO voting, Charity funds, Memory Book
  • Q3 2025: NFT icons, Pilgrim tracker, Parish support program
  • Q4 2025: Integration with real churches, partnerships, marketing campaign

7. Revenue Model

  • Token sales: Users buy tokens on DEX (Meteora, Jupiter)
  • Transaction fees: None (0% transfer fee) — users pay Solana gas fees
  • Future optional services: Premium confession (chat with a real priest), parish subscriptions
